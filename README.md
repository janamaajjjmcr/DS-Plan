# 🎓 Data Science Interactive Study Plan

This is an **experimental project** designed to transform a static university study plan into an **interactive roadmap** for students.  
It aims to help students easily track:

✅ The courses they have already completed  
📘 The courses they are currently taking  
🔓 The courses they are eligible to register for based on prerequisites

🔗 **Live Demo:**  
[👉 View the Interactive Plan](https://janamaajjjmcr.github.io/DS-Plan/)

---

## ✨ Features

- 📊 Visual, interactive course layout  
- 🔄 Dynamic display of prerequisites and course status  
- 🧭 Helps students navigate their academic journey
